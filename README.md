FileAPI for Yii2
===========================

Installation
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

```json
{
    ...

    "require": {
        "otsec/yii2-fileapi": "0.2.*"
    },

    "repositories": [
        {
            "type": "vcs",
            "url":  "https://github.com/otsec/yii2-fileapi.git"
        }
    ],
}
```
